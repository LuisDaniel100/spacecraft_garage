
*La aplicación web empieza en nuestro archivo "index.php" donde se podrá iniciar sesión y/o registrarse. 

Para el registro solicitará:
	-email.
	-nombre.
	-contraseña.
	-confirmación de contraseña.

* Una vez que tengamos nuestro usuario creado, podremos iniciar sesión con el correo y contraseña correspondiente. 

* Al iniciar sesión, se redirigirá a al archivo "dashboard.php" aquí nos enlistará las naves que se han registrado, tenemos el botón de cerrar sesión y el botón de crear una nave. al crear una nave se nos abrirá un formulario que contiene los campos:
	
	-nombre de la nave
	-nacionalidad
	-año de construcción
	-Foto
Al llenar esos datos podremos dar click en agregar y nos redirigirá al dashboard donde podremos visualizar que se añadió la nave. 

* Tenemos el botón "ver/editar" este botón se encuentra sin función, la idea es agregar un formulario que se llene automáticamente con los datos de ese registro, para poder hacer modificaciones directas en la nave que se desee.
